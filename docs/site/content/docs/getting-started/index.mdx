Turborepo를 처음 사용하는 경우 다음 단계를 따라 시작할 수 있습니다.

<Steps>
<Step>
## Turborepo 설치

리포지토리 어디에서나 터미널에서 `turbo` 명령을 편리하게 실행할 수 있도록 `turbo`를 전역으로 설치합니다.

<Tabs items={["npm", "yarn", "pnpm"]} storageKey="selected-pkg-manager">
  <Tab value="npm">
  ```bash title="Terminal"
  npm install turbo --global
  ```

  </Tab>
  <Tab value="yarn">
  ```bash title="Terminal"
  yarn global add turbo
  ```

  </Tab>
  <Tab value="pnpm">
  ```bash title="Terminal"
  pnpm add turbo --global
  ```

  </Tab>
</Tabs>

`turbo` 설치에 대해 자세히 알아보려면 [설치 가이드](/docs/getting-started/installation)를 참조하세요.

</Step>
