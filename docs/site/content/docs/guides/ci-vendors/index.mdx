## 일반 설정

CI에 대해 Remote Caching을 활성화하려면:

1. Remote Cache에 액세스하기 위해 CI에서 Turborepo에 대한 환경 변수를 설정합니다.
   | 변수 | 설명 |
   | ----------- | ----------- |
   | `TURBO_TOKEN` | Remote Cache에 액세스하기 위한 Bearer 토큰 |
   | `TURBO_TEAM` | 저장소와 연결된 계정 이름. <a href="https://vercel.com/docs/monorepos/remote-caching#vercel-remote-cache" rel="noreferrer noopener" target="_blank" >Vercel Remote Cache</a>를 사용할 때 이것은 [팀의 슬러그](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fsettings&title=Get+team+slug)입니다. |
2. 저장소를 복제합니다.
3. 패키지 매니저를 통해 의존성을 설치합니다.
4. `turbo`를 통해 작업을 실행합니다.
