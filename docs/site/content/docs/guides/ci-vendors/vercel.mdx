---
title: Vercel
description: Vercel에서 Turborepo를 사용하는 방법을 알아봅니다.
---

![](/images/docs/turborepo-x-vercel.png)

Vercel의 Turborepo와의 제로 구성 통합은 모노레포를 자동으로 이해합니다.

Vercel에 Turborepo를 배포하려면 [새 프로젝트를 생성](https://vercel.com/new)하고 코드를 가져옵니다. 프로젝트는 [Vercel Remote Cache](https://vercel.com/docs/monorepos/remote-caching)를 사용하기 위한 올바른 설정으로 사전 구성됩니다.

Turborepo를 Vercel에 배포하는 방법에 대한 자세한 내용은 [Vercel 문서를 방문](https://vercel.com/docs/concepts/monorepos/turborepo)하세요.
