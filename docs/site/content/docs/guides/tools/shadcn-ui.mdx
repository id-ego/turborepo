import { PackageManagerTabs, Tab } from '#components/tabs';

[shadcn/ui](https://ui.shadcn.com/docs/monorepo)는 Tailwind CSS로 만든 아름답게 디자인된 오픈 소스 컴포넌트 세트로, 앱에 복사하여 붙여넣을 수 있습니다.

새 monorepo에서 shadcn/ui를 시작하려면 다음을 실행하세요:

<PackageManagerTabs>

<Tab value="pnpm">

```bash title="Terminal"
pnpm dlx shadcn@canary init
```

</Tab>

<Tab value="yarn">

```bash title="Terminal"
npx shadcn@canary init
```

</Tab>

<Tab value="npm">

```bash title="Terminal"
npx shadcn@canary init
```

</Tab>

<Tab value="bun (Beta)">

```bash title="Terminal"
bunx shadcn@canary init
```

</Tab>
</PackageManagerTabs>

메시지가 표시되면 monorepos 옵션을 선택합니다.

컴포넌트를 추가하려면 다음을 실행하세요:

<PackageManagerTabs>

<Tab value="pnpm">

```bash title="Terminal"
pnpm dlx shadcn@canary add [COMPONENT]
```

</Tab>

<Tab value="yarn">

```bash title="Terminal"
npx shadcn@canary add [COMPONENT]
```

</Tab>

<Tab value="npm">

```bash title="Terminal"
npx shadcn@canary add [COMPONENT]
```

</Tab>

<Tab value="bun (Beta)">

```bash title="Terminal"
bunx shadcn@canary add [COMPONENT]
```

</Tab>
</PackageManagerTabs>
