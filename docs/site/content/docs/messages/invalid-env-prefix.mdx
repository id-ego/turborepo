---
title: 잘못된 환경 변수 접두사
description: Turborepo에서 잘못된 환경 변수 접두사 오류에 대해 자세히 알아보세요.
---

## 이 오류가 발생한 이유

`turbo.json`에서 환경 변수를 선언할 때 `$` 접두사를 사용할 수 없습니다. 이것은
Turborepo 1.5에서 더 이상 사용되지 않는 환경 변수 의존성 선언을 위한 오래된 구문이었습니다.

```json title="./turbo.json"
{
  "globalEnv": ["$MY_ENV_VAR"]
}
```

위에서 선언된 환경 변수는 `$` 접두사를 가지고 있습니다.

## 해결 방법

환경 변수 선언에서 `$` 접두사를 제거하세요.

```json title="./turbo.json"
{
  "globalEnv": ["MY_ENV_VAR"]
}
```

`npx @turbo/codemod migrate-env-var-dependencies`를 사용하여 `env` 및 `globalEnv` 키로 마이그레이션할 수 있습니다.
자세한 내용은 [codemod 문서](/docs/reference/turbo-codemod#turborepo-1x)를 확인하세요.
