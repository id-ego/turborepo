---

## 이 오류가 발생한 이유

단일 패키지 모드에서는 리포지토리에 여러 패키지가 있을 수 없습니다. 따라서 `turbo.json`에서
지정된 패키지 이름으로 작업을 선언하는 것은 허용되지 않습니다.

```json title="./turbo.json"
{
  "tasks": {
    "app#build": {
      "cache": true
    }
  }
}
```

## 해결 방법

작업 선언에서 패키지 이름을 제거하세요.

```json title="./turbo.json"
{
  "tasks": {
    "build": {
      "cache": true
    }
  }
}
```

또는 여러 패키지를 사용하려면 [리포지토리에서 워크스페이스를 지정](/docs/getting-started/add-to-existing-repository)할 수 있습니다.
