---

`turbo` 바이너리의 경로를 가져옵니다.

```bash title="Terminal"
turbo bin
```

[**전역 `turbo`**](/docs/getting-started/installation#global-installation)를 사용하는 경우, 이것은 전역 `turbo` 바이너리의 경로입니다. `turbo`를 설치하는 데 사용한 패키지 매니저의 전역 디렉토리 경로를 볼 수 있습니다.

[**로컬 `turbo`**](/docs/getting-started/installation#repository-installation)를 사용하는 경우, 이것은 로컬 `turbo` 바이너리의 경로입니다. `turbo`가 저장소에 설치된 경우, `node_modules`의 경로일 가능성이 높습니다.
