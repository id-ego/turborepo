저장소를 Remote Cache 제공자에 연결합니다.

```bash title="Terminal"
turbo link
```

선택한 소유자(사용자 또는 조직)는 [Remote Caching](/docs/core-concepts/remote-caching)을 통해 [캐시 아티팩트](/docs/core-concepts/remote-caching)를 공유할 수 있습니다.

## 플래그 옵션

### `--api <url>`

Remote Cache 제공자의 URL을 지정합니다.

```bash title="Terminal"
turbo link --api=https://example.com
```

### `--yes`

모든 프롬프트에 yes로 답변합니다

```bash title="Terminal"
turbo link --yes
```

### `--scope <scope>`

연결할 범위입니다. 예를 들어, Vercel을 사용할 때 이것은 Vercel 팀의 슬러그입니다.

```bash title="Terminal"
turbo link --scope=your-team
```
