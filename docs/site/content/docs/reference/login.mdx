Remote Cache 제공자에 로그인합니다.

```bash title="Terminal"
turbo login
```

기본 제공자는 [Vercel](https://vercel.com/)입니다. 다른 제공자를 지정하려면 `--api` 옵션을 사용하세요.

## 플래그 옵션

### --api \<url>

Remote Cache 제공자의 API URL을 설정합니다.

```bash title="Terminal"
turbo login --api=https://acme.com/api
```

### --login \<url>

토큰을 동적으로 생성해야 하는 로그인 요청의 URL을 설정합니다.

```bash title="Terminal"
turbo login --login=https://acme.com
```

### --sso-team \<team>

팀 슬러그를 제공하여 SSO가 활성화된 팀에 연결합니다.

```bash title="Terminal"
turbo login --sso-team=slug-for-team
```

### --manual

로그인 서비스에서 토큰을 요청하는 대신 수동으로 토큰을 입력합니다.
