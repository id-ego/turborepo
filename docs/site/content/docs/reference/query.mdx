모노레포에 대해 GraphQL 쿼리를 실행합니다.

```bash title="Terminal"
turbo query [query] [flags]
```

인수가 전달되지 않으면 명령어는 쿼리를 실행할 수 있는 GraphiQL playground를 엽니다.

```bash title="Terminal"
turbo query
```

쿼리 문자열이 전달되면 명령어는 쿼리를 실행하고 결과를 출력합니다.

```bash title="Terminal"
turbo query "query { packages { items { name } } }"
```

파일 경로가 전달되면 명령어는 파일을 읽고 쿼리를 실행합니다.

```bash title="Terminal"
turbo query query.gql
```
