## 작업의 환경 변수

Turborepo는 작업이 실행되는 동안 다음 환경 변수를 작업 내에서 사용할 수 있도록 합니다:

| 변수           | 설명                                                                                                                                     |
| -------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| `TURBO_HASH`   | 현재 실행 중인 작업의 해시입니다.                                                                                                        |
| `TURBO_IS_TUI` | [TUI](/docs/reference/configuration#ui)를 사용할 때 이 변수는 `true`로 설정됩니다.                                                       |
| `TURBO_IS_MFE` | [마이크로프론트엔드](/docs/guides/microfrontends)를 사용할 때 이 변수는 애플리케이션의 `microfrontends.json`에 정의된 포트로 설정됩니다. |
