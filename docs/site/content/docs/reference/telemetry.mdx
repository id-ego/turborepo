```bash title="Terminal"
turbo telemetry [argument]
```

이 머신의 텔레메트리를 관리합니다.

<LinkToDocumentation href="/docs/telemetry">
  Turborepo의 텔레메트리에 대해 자세히 알아보기.
</LinkToDocumentation>

## 인수

### `status`

이 머신의 현재 텔레메트리 상태를 조회합니다.

```bash title="Terminal"
turbo telemetry status
```

### `enable`

이 머신의 텔레메트리를 활성화합니다.

```bash title="Terminal"
turbo telemetry enable
```

### `disable`

이 머신의 텔레메트리를 비활성화합니다.

```bash title="Terminal"
turbo telemetry disable
```
